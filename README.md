# 🎓 Student Management System

##  Technologies Used
- Java (JSP, Servlets)
- JDBC
- MySQL
- Bootstrap

##  Features
- User Registration & Login
- CRUD Operations (Add, View, Update, Delete)
- Search Functionality
- Session Management
- Responsive UI

##  Architecture
- MVC Pattern (DAO, Servlet, JSP)

##  Setup Instructions
1. Import project into Eclipse
2. Add MySQL connector (JAR file)
3. Create database: `student_db`
4. Run on Apache Tomcat server
